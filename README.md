# Virtual-Mouse
[Demo](https://www.linkedin.com/posts/saimum-islam_today-i-played-around-a-bit-with-creating-activity-7403467520944353282-jO7b/)

I played around a bit with creating an AR-based virtual mouse system. I worked on hand-tracking using MediaPipe, PyAutoGUI, and started detecting hand landmarks through my webcam. The basic detection works, although I have faced a few issues, mainly with the implementation of scrolling.
